Be precise and conservative — only extract what is clearly present in the content. Do not infer or embellish.
---USER---

Content to analyze:
{{content}}

Extract the following fields:
- capture_type: One of: decision, idea, observation, task, win, blocker, question, reflection. Choose the single best fit.
- people: Array of person names explicitly mentioned (strings). Empty array if none.

- summary: One sentence (max 120 characters) capturing the essential point of this capture.
